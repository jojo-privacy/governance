This repository defines the governance, decision-making processes, and maintainer responsibilities for the **Jojo Privacy** organization.

Jojo Privacy develops privacy-sensitive and security-critical open-source software.  
As such, clear governance and strict decision processes are required to protect users, contributors, and the long-term integrity of the project.

This repository contains **no product code**.

---

## Purpose

The purpose of this repository is to document:

- How technical and non-technical decisions are made
- Who has authority to review and merge changes
- How maintainers are added or removed
- Expectations for professional and responsible participation

Governance exists to ensure that openness does not compromise security, privacy, or trust.

---

## Scope

The policies in this repository apply to:

- All repositories under the Jojo Privacy organization
- All maintainers and contributors
- All technical, security, and privacy-related decisions

Product-specific rules may exist in individual repositories, but they must not conflict with this governance.

---

## Guiding Principles

Jojo Privacy operates under the following principles:

- **Security over features**
- **Privacy over convenience**
- **Transparency over speed**
- **Long-term trust over short-term growth**

If a proposal violates these principles, it will not be accepted.

---

## Repository Structure

This repository includes the following documents:

- `DECISION_PROCESS.md`  
  Defines how decisions are proposed, discussed, reviewed, and finalized.

- `MAINTAINERS.md`  
  Defines maintainer roles, responsibilities, and expectations.

- `CODE_OF_CONDUCT.md`  
  Defines acceptable behavior and community standards.

Each document should be read and understood before contributing to any Jojo Privacy project.

---

## Changes to Governance

Changes to governance policies require:
- Clear justification
- Public discussion
- Approval by maintainers

Governance changes are treated as high-impact decisions and are reviewed carefully.

---

## Transparency and Accountability

Jojo Privacy aims to keep governance transparent and publicly accessible.

In cases involving security vulnerabilities or responsible disclosure, details may be temporarily withheld until risks are mitigated.

---

## Contact

For questions related to governance, contact the maintainers via GitHub discussions or issues in this repository.

Security-related matters should follow the disclosure process defined in the `security` repository.
